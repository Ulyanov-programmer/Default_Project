Шрифты

Эта сборка умеет работать со шрифтами. Ещё бы она не умела.

Конвертация в woff2

Для того что бы конвертировать шрифт, достаточно добавить его в любое место папки для исходников, кроме ассетов. Обычно в sources/fonts/

Добавленые шрифты будут конвертированы из форматов .otf и .ttf в .woff2

Автогенерация @font-face

Один из скриптов занимается тем, что записывает @font-face автоматически. Он находится по пути plugins/fontsWriting.js и записывает результат в sources/styles/fonts.pcss

Например, он преобразует Montserrat-VariableFont_wght.ttf в это: 

@font-face {
  font-style: normal;
  font-weight: 100 1000;
  src: url("../fonts/Montserrat-VariableFont_wght.woff2") format("woff2-variations");
  font-family: "Montserrat";
  font-display: swap;
} 

:root {
  --montserrat: 'Montserrat';
}

.Montserrat {
  font-family: var(--montserrat);
  font-weight: 100 1000;
  font-style: normal;
}
