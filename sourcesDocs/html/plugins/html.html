HTML

Эта сборка умеет работать с HTML. В первую очередь с PostHTML.

В принципе вам ничто не мешает просто писать HTML-файлы, но для упрощения разработки они по умолчанию обрабатываются PostHTML - постпроцессором для HTML. Он преобразует файлы, выполняя над ними ту работу, которую не хотелось бы делать вручную.

Для того, что бы работать с HTML-файлами, создайте их в любом месте, кроме sources/assets/ и sources/components/ (файлы оттуда можно использовать как шаблоны).

Они будут перемещены в dist/ с сохранением структуры папок (например, если файл был в sources/html/pages/ он будет перемещён в dist/html/pages/).

Вы сможете найти файл конфигурации для HTML по пути sources/plugins/posthtml.js (posthtml обрабатывает все html-файлы по умолчанию).
