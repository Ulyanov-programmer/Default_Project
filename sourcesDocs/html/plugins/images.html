Изображения

Эта сборка умеет обрабатывать изображения форматов:

  • gif
  • webp
  • avif
  • png
  • jpg | jpeg
  • svg(пропускает без изменений)

Это обеспечивает sharp.

Для того, что бы обработать изображения, создайте/переместите их в любое место, кроме sources/assets/

Используйте гибкий API плагина, что бы преобразовать изображения так, как вам нужно. Их обрабаотанные версии будут перемещены в dist/ с сохранением структуры папок (например, если файл был в sources/images/subfolder/ он будет перемещён в dist/images/subfolder/).

Вы сможете найти файл конфигурации по пути sources/plugins/sharp.js
